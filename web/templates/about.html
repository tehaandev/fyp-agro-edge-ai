About This Project

Edge-Based ML for Smart Irrigation

Real-time decision support system for small-scale farms using lightweight machine learning models optimized for Raspberry Pi 4B deployment.

Project Goals
  • Enable offline ML inference on resource-constrained edge devices
  • Provide smart irrigation recommendations based on sensor data
  • Detect plant diseases from leaf images
  • Operate completely offline - no internet required
  • Target smallholder farms in Sri Lanka (<2 hectares)
Technical Specifications
Hardware:
  • Raspberry Pi 4B (4GB RAM)
  • DHT11 sensor (temperature, humidity)
  • Soil moisture sensor
  • Pi Camera (optional)
Models:
  • Irrigation: RandomForest (50KB, 97.89% acc)
  • Disease: MobileNetV2 TFLite (9.1MB, 98% acc)
  • Total inference: <200ms
  • Power consumption: <5W
ML Pipeline
  1. Data Collection: Hourly sensor readings (temp, humidity, soil moisture)
  2. Feature Engineering: 37 features (rolling stats, change rates, time features)
  3. Model Training: Jupyter notebooks with automated logging
  4. Edge Deployment: TFLite and pickle models for offline inference
  5. Decision Support: Flask web interface for farmers
Target Users

Smallholder farmers in Sri Lanka with limited access to:

  • Reliable internet connectivity
  • Technical expertise
  • Agricultural extension services
Performance Targets (Achieved)
  • ✅ Inference time: <1 second
  • ✅ Model size: <50MB per model
  • ✅ Accuracy: >95%
  • ✅ Power: <5W
